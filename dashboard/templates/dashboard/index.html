How It Works
Spot Mode
• Signal fires → Market BUY immediately
• OCO order placed: TP1 limit + SL stop-limit
• TP2 limit order for 35% of position
• TP3 limit order for 25% of position
• SL triggers automatically — no monitoring needed
• SELL signals are skipped (no shorting on spot)
Futures Mode
• Signal fires → Sets leverage (2–3x conservative)
• Market LONG (BUY) or SHORT (SELL)
• 3x TAKE_PROFIT_MARKET orders at TP1/TP2/TP3
• STOP_MARKET closes full position at SL
• Uses MARK_PRICE to avoid wick manipulation
• Hedge mode: can hold LONG and SHORT same time
Setup
1. Get Binance API keys at Binance → Profile → API Management → Create API
2. Enable "Spot & Margin Trading" for spot mode. Enable "Futures Trading" for futures. Never enable withdrawals.
3. Add to your .env file:
BINANCE_API_KEY=your_key_here BINANCE_API_SECRET=your_secret_here AUTO_RISK_PCT=2.0 AUTO_LOSS_LIMIT=6.0 4. By default runs in TESTNET mode — orders go to testnet.binance.vision, not real money.
5. Add BINANCE_TESTNET=false to go live. Test for at least 1 week first.

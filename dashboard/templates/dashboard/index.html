Setup
1. Binance → Profile → API Management → Create API key
2. Enable Spot Trading for spot · Enable Futures Trading for futures · Never enable withdrawals
3. Add to .env: BINANCE_API_KEY=your_key BINANCE_API_SECRET=your_secret BINANCE_TESTNET=true 4. Default: Testnet mode — safe to test. Set BINANCE_TESTNET=false to go live.
